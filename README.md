# Modern Portfolio

A modern, professional portfolio website designed for developers and creatives to showcase their work. This project features a clean, responsive design with smooth animations and bilingual support (English/Chinese).

## Features

- **Modern Design**: Clean and professional UI with gradient accents and smooth animations
- **Responsive Layout**: Fully responsive design that works seamlessly on desktop, tablet, and mobile devices
- **Bilingual Support**: Built-in English and Chinese language switching with local storage persistence
- **Interactive Elements**: Smooth scrolling, hover effects, and scroll-triggered animations
- **Contact Form**: Functional contact form with validation
- **Mobile Navigation**: Hamburger menu for mobile devices

## Project Structure

```
├── .gitattributes      # Git configuration
├── README.md           # English documentation
├── README.zh-CN.md     # Chinese documentation
├── index.html          # Main HTML file
└── assets/
    ├── styles.css      # All CSS styles
    └── script.js       # JavaScript functionality and translations
```

## Technologies Used

- **HTML5**: Semantic markup for better accessibility and SEO
- **CSS3**: Modern styling with CSS Grid, Flexbox, and custom properties
- **JavaScript**: Vanilla JS for interactivity and internationalization
- **Google Fonts**: Inter font family for clean typography

## Sections

1. **Hero**: Introduction with profile image, social links, and call-to-action buttons
2. **About**: Personal journey timeline and skills showcase
3. **Projects**: Portfolio grid with project cards and technology tags
4. **Contact**: Contact information and functional message form
5. **Footer**: Quick links and social media connections

## Customization

You can easily customize this portfolio by:

- Updating the content in `index.html`
- Modifying colors and styles in `assets/styles.css`
- Adding more translations in `assets/script.js`
- Replacing placeholder images with your own
