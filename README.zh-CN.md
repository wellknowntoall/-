# 现代化作品集网站

一个现代化的专业作品集网站，专为开发者和创意工作者设计，用于展示他们的作品。该项目具有简洁、响应式的设计，流畅的动画效果，并支持中英双语。

## 功能特性

- **现代设计**：简洁专业的 UI，采用渐变配色和流畅动画
- **响应式布局**：完全响应式设计，在桌面、平板和移动设备上都能完美呈现
- **双语支持**：内置中英文语言切换功能，支持本地存储记忆
- **交互元素**：平滑滚动、悬停效果和滚动触发动画
- **联系表单**：带验证功能的实用联系表单
- **移动端导航**：移动设备的汉堡菜单

## 项目结构

```
├── .gitattributes      # Git 配置
├── README.md           # 英文项目文档
├── README.zh-CN.md     # 中文项目文档
├── index.html          # 主 HTML 文件
└── assets/
    ├── styles.css      # 所有 CSS 样式
    └── script.js       # JavaScript 功能和翻译
```

## 技术栈

- **HTML5**：语义化标签，更好的可访问性和 SEO
- **CSS3**：现代样式，使用 CSS Grid、Flexbox 和自定义属性
- **JavaScript**：原生 JS 实现交互和国际化
- **Google Fonts**：Inter 字体家族，简洁排版

## 页面板块

1. **首页**：个人简介、头像、社交链接和行动按钮
2. **关于**：个人经历时间线和技能展示
3. **项目**：作品集网格，包含项目卡片和技术标签
4. **联系**：联系信息和功能完整的留言表单
5. **页脚**：快速链接和社交媒体连接

## 自定义

你可以轻松自定义这个作品集：

- 在 `index.html` 中更新内容
- 在 `assets/styles.css` 中修改颜色和样式
- 在 `assets/script.js` 中添加更多翻译
- 替换占位图片为你自己的图片
